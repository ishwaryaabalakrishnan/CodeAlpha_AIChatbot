# AI Chatbot

## Project Description
AI Chatbot is a Java-based console application developed as part of the CodeAlpha Java Programming Internship. It is a rule-based chatbot that interacts with users by responding to common questions using predefined answers.

## Features
- Greets users
- Answers common questions
- Responds to Java and OOP related queries
- Interactive console-based chatbot
- Exit command to end the conversation
- User-friendly interface

## Technologies Used
- Java
- IntelliJ IDEA
- Scanner Class
- Object-Oriented Programming (OOP)

## How to Run
1. Open the project in IntelliJ IDEA.
2. Run the `AIChatbot.java` file.
3. Type your questions in the console.
4. Type `bye` to exit the chatbot.

## Sample Conversation

```
==================================
        AI CHATBOT
==================================
Bot: Hello! I am your AI Chatbot.
Bot: Type 'bye' to exit.

You: hello
Bot: Hello! Nice to meet you.

You: what is java
Bot: Java is an object-oriented programming language.

You: thank you
Bot: You're welcome!

You: bye
Bot: Goodbye! Have a nice day.
```
